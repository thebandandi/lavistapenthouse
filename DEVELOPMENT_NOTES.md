Technical Anchor: La Vista PenthouseProject Overview: A direct-booking, high-end vacation rental website for a penthouse in Cabo/Pescadero, built with a "headless" architecture to compete with OTAs (Airbnb/VRBO).1. Core Tech StackFramework: Vanilla HTML5/JavaScript (optimized for speed).Hosting: Netlify (utilizing Serverless Functions & Netlify Blobs for persistent storage).CI/CD: GitHub repository with automated deployment to Netlify.DNS/Security: Cloudflare (Domain: lavistapenthouse.com).Local Dev: Netlify CLI on Windows (PC) using netlify dev and .env files for local key management.2. Media & Content ManagementProperty Imagery (Cloudinary): All high-resolution penthouse photos are hosted and served via Cloudinary for dynamic device-based optimization and fast load times.Stock Content (Pexels API): The blog generator uses the Pexels API to programmatically source high-quality, royalty-free atmospheric images matching AI-generated search terms.AI Content (Google Gemini): Automated, bilingual, researched blog generation powered by the Gemini 2.5 API with search-tool capabilities.3. Operational IntegrationsLead Generation (Netlify Forms): Built-in serverless form handling for the "Contact Us" section with automatic email notifications and spam protection.Payments: Stripe (Secure direct-booking checkout logic for direct revenue).Calendar Sync: Real-time availability protection using bi-directional iCal feeds from Airbnb and VRBO.Pricing Strategy: Transitioning from Fixed Rate to Dynamic Pricing (Target: DPGO 0.5% model synced via Airbnb bridge).4. Key Environment VariablesGEMINI_API_KEY: Content research and generation.PEXELS_API_KEY: Image sourcing for blog posts.STRIPE_SECRET_KEY: Payment processing security.BLOG_ADMIN_PASSWORD: Access control for the /admin dashboard.5.
